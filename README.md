### Northern Ireland Crime Analysis (2016–2019)
This project analyses police‑recorded crime data from Northern Ireland between 2016 and 2019. The aim is to identify crime trends, high‑risk areas, and patterns across different crime categories using descriptive and predictive analytical techniques. The work forms part of an academic assessment and demonstrates practical skills in data preparation, statistical analysis, time‑series exploration, and clustering.

### Project Overview
The analysis focuses on four key objectives:

Identifying annual crime trends across Northern Ireland

Determining which cities recorded the highest crime levels

Highlighting the most frequently reported crime types

Grouping cities and crime categories using clustering techniques to uncover similarities and differences

The project uses a combination of descriptive analytics (correlation, trend analysis) and predictive analytics (time‑series analysis, K‑means clustering) to extract meaningful insights from the dataset.

### Dataset
The dataset was sourced from data.world, containing police‑recorded crime data for Northern Ireland.
It includes:

Monthly and yearly crime counts

Crime outcomes

Crime categories

City/district‑level breakdowns

The raw dataset originally spanned 2001–2019, but this project focuses on 2016–2019 to allow for a more detailed and manageable analysis.

### Methodology
1. Data Preparation
Extracted relevant years (2016–2019)

Separated mixed numeric values into individual columns

Summed monthly crime counts into annual totals

Standardised numeric values for clustering

Cleaned and structured the dataset for analysis

2. Descriptive Analytics
Correlation analysis to examine the relationship between crime counts and outcomes

Scatter plots to visualise linear relationships

Trend analysis to observe year‑on‑year changes

3. Predictive Analytics
Time‑series analysis to identify cities and crime types with the highest recorded activity

K‑means clustering to group similar cities and crime categories based on crime patterns

### Key Findings
Crime Trends
A strong positive correlation (0.96) was found between crime counts and outcomes, indicating consistent reporting and resolution patterns.

Highest Crime City
Belfast City consistently recorded the highest number of crimes across all four years.

Most Common Crime Types
Two crime categories appeared most frequently:

Violence without injury (including harassment)

Criminal damage / domestic burglary

Clustering Insights
K‑means clustering revealed:

Mid Ulster and Mid & East Antrim frequently grouped together due to similar crime levels

For crime types, Theft – domestic burglary and Theft – vehicle offences formed consistent centroids
